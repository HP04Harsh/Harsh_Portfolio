🌐 Harsh Pardhi | Personal Portfolio
====================================

Welcome to the repository for my professional portfolio. This project is a modern, responsive web application designed to showcase my journey as a **Software Engineer** and **Platform Architect**, focusing on Cloud-Native technologies, Infrastructure as Code, and CI/CD.

✨ Live Demo
-----------

You can view the live site here: [**Harsh Portfolio**](https://hp04harsh.github.io/Harsh_Portfolio/)

🚀 Features
-----------

*   **Modern Glassmorphism UI:** A clean, white-themed aesthetic with soft transparency and blur effects.
    
*   **DevOps Focused:** Highlights core competencies in Azure, Kubernetes, Terraform, and Go/Python.
    
*   **Fully Responsive:** Optimized for a seamless experience across mobile, tablet, and desktop devices.
    
*   **Automated Deployment:** Integrated with **GitHub Actions** for continuous deployment to GitHub Pages.
    

🛠️ Tech Stack
--------------

*   **HTML5 & CSS3:** Semantic structure and custom styling using modern CSS variables.
    
*   **JavaScript:** Minimalist logic for interactive UI elements.
    
*   **GitHub Actions:** Automated CI/CD pipeline for static site hosting.
    
*   **Vercel/GitHub Pages:** High-performance hosting environments.
    

📂 Project Structure
--------------------
```
Harsh_Portfolio/
├── .github/workflows/  # CI/CD pipeline (Azure/GitHub Pages)
├── assets/             # Images, icons, and styling resources
├── index.html          # Main portfolio entry point
└── .gitignore          # Build and environment exclusions
```
